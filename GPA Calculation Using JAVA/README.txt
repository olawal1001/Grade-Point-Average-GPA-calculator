**GPA Calculator**
This Java program calculates a studentâ€™s Grade Point Average (GPA) based on grades from their classes. It provides functionality to compute both the semester GPA and the cumulative GPA.

**Features**
Input: The program accepts course information such as course name, credit hours, and letter grade.
Calculation: It computes the GPA using the standard GPA scale (A=4.0, B=3.0, etc.).
Graphical User Interface (GUI): The program includes a simple GUI for user interaction.
Display: The calculated GPA is displayed to the user.
Usage
Compile the Java files:
javac GpaCalculatorApp.java

**Run the program:**
java GpaCalculatorApp

**Input Course Details:**
Enter the course name.
Specify the credit hours.
Choose the letter grade (A, B, C, etc.).
View GPA:
The program will display the calculated GPA.
